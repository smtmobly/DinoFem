# DinoFem 

版本号：v0.3.0

## 1 依赖库
(1) pyvista '0.27.2'

使用命令pip install pyvista 安装

(2) numpy 1.19.4

使用命令pip install numpy 安装

## 2 文档查看

在终端使用
```
python -m pydoc -p 8899
```
命令即打开本地服务器查看
